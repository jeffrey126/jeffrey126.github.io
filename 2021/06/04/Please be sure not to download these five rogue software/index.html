    请一定不要这五款流氓软件，另推荐免费好用软件。

    发表于 2021-06-04 分类于 windows ， software ， 软件 阅读次数：
    本文字数： 1.1k 阅读时长 ≈ 1 分钟

    以下五款流氓软件，请勿下载。

    1.2345浏览器
    2.金山毒霸
    3.驱动精灵
    4.搜狗输入法
    5.360安全卫士
    安装完这些软件，你就知道你的电脑会变得有多卡了。

    替代方案：

    1.谷歌浏览器，火狐浏览器
    2.火绒安全
    3.驱动人生
    4.讯飞输入法

    最值得推荐的五大精品软件

    1.火绒安全（免费） -杀毒软件

    版本类型：
    精简版
    完整版（推荐下载）

    功能简介：
    这款杀毒软件，内含了18个重要的防护功能，可以有效的查杀病毒、木马删除流氓软件，拦截恶意网站等。平时除了出现更新病毒库提示之外，再没有其他任何的打扰，并且它的弹窗拦截功能非常的好用。

    2.recuva professional

    免费的文件恢复工具
    版本类型：
    免费-自由
    专业版
    专业套装版本

    功能简介：
    不仅支持本地磁盘的文件恢复，还支持储存卡、外部硬盘、U盘等位置的数据恢复。
    如果你想永久删除某个文件的话，那么你也可以通过这款软件来实现。
    这款软件的安全覆盖功能，使用的是军事级别的删除技术，可以确保您的文件被彻底的删除。
    对于个人来说，它的免费版已经足够使用了。

    3.Potplayer(万能的视频播放器)

    非常好用的一个功能就是在线添加直播源，可以观看各大卫视。
    建立各大卫视方法如下：
    （打开软件→点击播放列表→点击新建专辑→8000直播源→外部播放→选项按键→选择地址→输入名称https://iptv-org.github.io/iptv/index.m3u→点击确定）
    还有很好用的视频录制功能。

    4.Bleachbit (免费开源磁盘清理软件)

    版本类型：
    windows
    linux
    windows用户，下载安装第一个安装包。
    功能简介：
    通过这款软件可以更轻松，更深层的完成磁盘清理，帮我们节省更多的宝贵时间。
    它主要的功能有释放缓存，删除cookie，清除internet历史记录，删除临时文件，删除日志，并清除我们的上网浏览记录。
    ps:深度搜索的时候，点击取消不然非常耗时间。

    5.FDM 免费下载软件

    市面上最好用的是IDM下载软件，但是收费，其破解版可用，如果怕有风险，不想用破解版，强烈建议使用FDM这款免费开源的下载软件。
    功能介绍：
    一款好的下载软件需要具备多种下载方式，它现在目前支持HTTP、FTP、BT种子、磁力链接等方式的下载。另外一点就是下载速度，这款下载工具它和迅雷之类的是一样的，下载的时候它会自动将文件切割成多个区块，同时进行多线程的下载，可以显著提高下载速度，并且它支持断点续传，不限速，界面也非常清爽，没有任何的广告。它和IDM一样可以通过浏览器进行无缝集成。
    PS:强烈建议有条件的人可以使用IDM付费软件，没有条件的人可以用FDM也是一样的。
